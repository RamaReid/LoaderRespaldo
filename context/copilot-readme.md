# Guía rápida para Copilot — GD Arquitectura

- Este proyecto usa un sistema cromático dual: histórico (print) y digital (web).
- Para TODO código: usar únicamente los colores del sistema digital.
- No inventar interfaces ajenas a arquitectura, diseño técnico, o estética constructiva.
- Cualquier componente nuevo debe:
  - respetar proporciones
  - mantener sobriedad
  - evitar animaciones exageradas
  - mantener estética racionalista/tectónica

Siempre preferir:
- fondo dark
- líneas técnicas claras
- azul D digital optimizado
- rojo G técnico

Nunca usar:
- blanco de fondo
- azul de fondo
- D blanca en animación
